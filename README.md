## 📦 投稿素材

👉 不会用 Git？点这里直接投稿：  
进入 Issues → 点击「提交素材」
---

## 🚀 如何贡献素材（进阶）

如果你会使用 GitHub：

1. Fork 本仓库
2. 上传素材到对应目录
3. 提交 Pull Request
4. 等待审核合并

---

## 📁 仓库结构

assets/
- images/        # 图片素材
- audio/         # 背景音乐
- sfx/           # 音效（UI音效等）
- others/        # 其他资源

---

## 📌 素材规范

### 🗂 文件命名
- 使用英文或拼音
- 避免空格（建议用 `-` 或 `_`）

### 📦 支持格式
- 图片：png / jpg / webp
- 音频：mp3 / wav
- 视频：mp4
- 压缩包：zip

### ⚠️ 大小限制
- 单文件建议 < 100MB
- 大文件请使用网盘链接
